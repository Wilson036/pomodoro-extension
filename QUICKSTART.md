# 🚀 番茄鐘插件 - 快速入門指南

這是一個簡化的安裝指南，讓你在 5 分鐘內開始使用番茄鐘插件！

## 📋 準備工作

確保你已安裝：
- ✅ Node.js (v16 或更高版本) - [下載連結](https://nodejs.org/)
- ✅ Chrome 瀏覽器

檢查 Node.js 版本：
```bash
node --version
npm --version
```

## 🎯 安裝步驟（只需 4 步）

### 步驟 1：進入專案目錄

```bash
cd pomodoro-extension
```

### 步驟 2：安裝依賴

```bash
npm install
```

⏱️ 需要約 30-60 秒，請耐心等待...

### 步驟 3：建構插件

```bash
npm run build
```

⏱️ 需要約 10-20 秒

✅ 完成後會看到 `dist` 資料夾

### 步驟 4：載入到 Chrome

1. 打開 Chrome，在網址列輸入：
   ```
   chrome://extensions/
   ```

2. 開啟右上角的 **「開發人員模式」** 開關

3. 點擊 **「載入未封裝項目」** 按鈕

4. 選擇專案中的 **`dist`** 資料夾

5. 完成！🎉 番茄鐘圖標會出現在工具列

## 🎮 開始使用

1. 點擊工具列的番茄鐘圖標
2. 設定你的循環次數和時間
3. 點擊「開始番茄鐘」
4. 專注工作！

## ⚙️ 開發模式（進階）

如果你想修改代碼：

```bash
# 啟動監聽模式
npm run dev

# 修改代碼後，在 Chrome 擴展頁面點擊「重新載入」
```

## ❓ 遇到問題？

### 問題 1：npm install 失敗

```bash
# 清理並重新安裝
rm -rf node_modules package-lock.json
npm install
```

### 問題 2：build 失敗

確認 Node.js 版本：
```bash
node --version  # 應該是 v16 或更高
```

### 問題 3：插件無法載入

- ✅ 確認已開啟「開發人員模式」
- ✅ 確認選擇的是 `dist` 資料夾
- ✅ 確認 `dist` 資料夾內有文件

### 問題 4：通知不顯示

- 檢查 Chrome 通知權限
- 檢查系統通知設定

## 🎨 自訂圖標（可選）

如果你想要自訂插件圖標：

1. 準備三個 PNG 圖標：
   - `icon16.png` (16x16 px)
   - `icon48.png` (48x48 px)
   - `icon128.png` (128x128 px)

2. 放到 `public/icons/` 目錄

3. 重新建構：
   ```bash
   npm run build
   ```

4. 在 Chrome 擴展頁面點擊「重新載入」

## 💡 使用技巧

### 番茄工作法建議

- **專心時間**：建議 25 分鐘（可根據任務調整）
- **休息時間**：建議 5 分鐘
- **循環次數**：建議 4 次後長休息
- **長休息**：15-30 分鐘

### 最佳實踐

1. 在開始前列出要完成的任務
2. 專心時間內關閉所有干擾
3. 休息時間真的要休息（離開螢幕）
4. 記錄完成的番茄鐘數量
5. 定期回顧和調整
